# 给小朋友看的微积分故事机

这是一个基于 `Python + Flask + GeoGebra` 的本地演示工具。

它不是标准的“微积分教材演示器”，而是一个更像故事书的启蒙版页面：先让孩子看懂感觉，再慢慢接近数学。

它从孩子能理解的体验开始讲：

- 山坡哪里陡，哪里平
- 很多小块块拼起来会变成一整片
- 现在下雨多快，会让水桶涨多快
- 骑车时每一刻的速度加起来，会得到总路程
- 函数图像和横轴围成的弯边面积，可以用积分计算

这些故事分别对应微积分里最核心的直觉：

- 局部变化
- 小量累加
- 当前变化速度和累计总量的关系
- 用积分解决实际问题
- 曲线下面积和定积分的关系

## 启动方式

1. 安装依赖

```bash
pip install -r requirements.txt
```

2. 启动服务

```bash
python app.py
```

3. 打开浏览器访问

```text
http://127.0.0.1:5000
```

面积极限单独教程：

```text
http://127.0.0.1:5000/area-limit
```

求导与不定积分教程：

```text
http://127.0.0.1:5000/derivative-integral
```

极值教程：

```text
http://127.0.0.1:5000/extrema
```

## 结构说明

- `app.py`: Flask 入口
- `templates/index.html`: 页面结构与 GeoGebra 容器
- `templates/area_limit.html`: 面积极限单独教程页面
- `templates/derivative_integral.html`: 求导与不定积分教程页面
- `templates/extrema.html`: 极值教程页面
- `static/style.css`: 儿童化界面样式
- `static/app.js`: 五个故事场景、GeoGebra API 交互、参数面板
- `static/area-limit.js`: 面积极限教程的五幕逻辑
- `static/area-limit.css`: 面积极限教程样式
- `static/derivative-integral.js`: 求导与不定积分教程的八幕逻辑
- `static/derivative-integral.css`: 求导与不定积分教程样式
- `static/extrema.js`: 极值教程的五幕逻辑
- `static/extrema.css`: 极值教程样式

## 建议怎么给孩子讲

1. 先讲“山坡”，不要说“导数”，只说“这里更陡”
2. 再讲“小块块拼草地”，让孩子接受“很多小东西加起来会变成整体”
3. 最后讲“雨水桶”，让孩子看到“现在快，累计也会涨得快”
4. 再讲“骑车去公园”，告诉孩子：如果知道每一刻骑得多快，就能算一共走了多远
5. 进入“曲线下面积”，把难度提高到初中级别：有限分段只能估算，积分真正计算的是分段无限变细时的极限面积，并可用原函数 `F(b)-F(a)` 精确求出
6. 如果孩子已经接受了这些故事，再告诉他：长大以后，这些感觉会有正式名字，叫微积分

## 这个版本不追求什么

- 不追求完整术语
- 不追求公式推导
- 不追求一次讲懂整个微积分

它追求的是：让一个孩子先不害怕，并且愿意说出“我好像明白了一点点”。
